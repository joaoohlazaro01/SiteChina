
Projeto Web

Este projeto é um site desenvolvido com **HTML, CSS e Bootstrap**, com o objetivo de apresentar informações sobre a China de forma moderna, organizada e visualmente atrativa.

---

## 🌐 Demonstração

👉 Acesse o site online:  
https://joaoohlazaro01.github.io/SiteChina

---

## 📚 Conteúdo do Site

O site é composto por várias páginas:

### 🏠 Home (Início)
- Introdução sobre a China
- Destaque histórico e econômico
- Cards de navegação

### 🍜 Culinária
- Pratos típicos chineses
- Yakisoba, Arroz Frito e Dim Sum
- Layout com cards e imagens

### 🎎 Cultura
- Tradições milenares
- Festivais, artes marciais e caligrafia
- Estrutura visual organizada com Bootstrap

### 🌏 Explore
- Pontos turísticos importantes
- Grande Muralha, Pequim e Xangai
- Foco em turismo e paisagens

### 📖 Sobre
- Página diferenciada com design moderno
- Banner (hero section)
- Estatísticas e informações relevantes
- Layout mais profissional

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- Bootstrap 5  
- Git e GitHub

## 🎨 Recursos e Funcionalidades

- Layout responsivo (mobile e desktop)
- Navbar personalizada com Bootstrap
- Cards organizados com sombras e estilos
- Hero section com imagem de fundo
- Design moderno e limpo
- Organização por seções
- Uso de imagens ilustrativas
